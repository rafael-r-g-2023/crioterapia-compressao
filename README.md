# Sistema de Crioterapia com Compressão

## Visão Geral

Projeto de desenvolvimento de um sistema portátil de crioterapia com compressão, inspirado em soluções existentes no mercado internacional.

O objetivo é desenvolver um produto funcional, com custo reduzido, voltado inicialmente para uso esportivo e recuperação muscular, com possibilidade de evolução futura para aplicações mais avançadas.

---

## Objetivos do Projeto

* Desenvolver um sistema integrado de:

  * resfriamento por água
  * compressão pneumática
* Criar um protótipo funcional (V1)
* Validar funcionamento através de testes experimentais
* Validar parâmetros de uso com especialista da área da saúde
* Reduzir custo em relação a soluções importadas

---

## Referências de mercado

* Breg VPULSE

## Metodologia

O projeto está sendo desenvolvido em etapas:

1. Engenharia reversa
   Análise de produtos existentes para entender funcionamento e arquitetura

2. Base científica
   Levantamento de limites de segurança e aplicação terapêutica

3. Testes experimentais
   Medição de vazão, pressão e temperatura

4. Validação com especialista
   Definição de parâmetros seguros de uso

5. Desenvolvimento do protótipo
   Construção e validação do sistema próprio

---

## Estratégia do Projeto

O projeto busca se posicionar entre:

* sistemas médicos de alto custo
* soluções esportivas de baixo custo

Proposta:

* portátil
* funcional
* custo reduzido
* modular
* escalável
