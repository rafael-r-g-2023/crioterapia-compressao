# Sistema de Crioterapia com Compressão

## Descrição
Projeto de desenvolvimento de um equipamento portátil para terapia de frio e compressão pneumática, com aplicação inicial para joelho.

## Objetivo
Criar um protótipo funcional (V1) baseado em engenharia reversa de sistemas existentes.

## Estrutura do Projeto

- docs/ → documentação técnica
- cad/ → modelos 3D
- eletronica/ → esquemas e PCB
- firmware/ → código

## Status
Em desenvolvimento — fase de arquitetura e definição de funcionamento
