# PLANILHAS DE TESTE

## Objetivo

Esta pasta contém planilhas utilizadas para testes e validação do sistema de crioterapia com compressão.

---

## Planilha de testes V1

Arquivo:
- planilha_testes_crioterapia_v1.xlsx

---

## Descrição

A planilha foi desenvolvida para registrar e analisar dados obtidos durante:

- Engenharia reversa do sistema de referência
- Testes do protótipo
- Validação hidráulica
- Validação pneumática
- Análise térmica

---

## Estrutura da planilha

### 1. Teste hidráulico
- Medição de vazão
- Cálculo automático de L/min

### 2. Teste pneumático
- Tempo de inflar
- Tempo de desinflar
- Observações

### 3. Teste térmico
- Temperatura de entrada
- Temperatura de saída

### 4. Teste por aplicador
- Comparação entre regiões (joelho, quadril, etc.)

---

## Como utilizar

1. Realizar os testes conforme documento:
   - docs/13_engenharia_reversa_experimental.md
   - docs/14_validacao_interface_prototipo.md

2. Inserir os dados medidos nas células indicadas

3. Utilizar os resultados para:
   - Definir componentes do sistema
   - Validar desempenho
   - Ajustar projeto

---

## Observações

- Esta planilha é uma ferramenta de apoio
- Os dados devem ser registrados com precisão
- Os testes devem ser realizados em bancada antes de uso real
