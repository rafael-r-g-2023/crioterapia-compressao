# APLICADORES TERAPÊUTICOS

## Objetivo

Definir o conceito de aplicadores intercambiáveis no sistema de crioterapia com compressão.

---

## Conceito do sistema

O produto é composto por:

- Unidade base (controle, bombeamento, compressão)
- Aplicador terapêutico (interface com o corpo)

---

## Função do aplicador

O aplicador é responsável por:

- Transferir frio para o corpo
- Aplicar compressão pneumática
- Adaptar-se à anatomia do usuário

---

## Estrutura do aplicador

O aplicador deve possuir:

- Canal de circulação de água
- Câmara pneumática
- Material flexível e confortável
- Sistema de fixação (velcro, cintas)

---

## Tipos de aplicadores

O sistema pode ser expandido para diferentes regiões:

- Joelho (V1)
- Quadril
- Ombro
- Tornozelo
- Coxa

---

## Impacto no sistema

Cada aplicador influencia diretamente:

- Transferência de calor
- Distribuição de pressão
- Eficiência do tratamento

---

## Considerações térmicas

- Diferentes regiões do corpo respondem de forma distinta ao frio
- Espessura de tecido influencia o resfriamento
- Contato com a pele impacta a eficiência

---

## Considerações pneumáticas

- Volume interno varia entre aplicadores
- Pressão percebida varia conforme área
- Necessidade de ajuste de ciclos

---

## Estratégia do projeto

- V1 focado exclusivamente em joelho
- Sistema projetado para futura expansão
- Arquitetura modular desde o início

---

## Aplicação em quadril

O uso no quadril é comum em contextos pós-operatórios, especialmente após cirurgias ortopédicas.

### Características

- Maior área de aplicação
- Maior espessura de tecido
- Maior dificuldade de transferência térmica

### Implicações para o projeto

- Pode exigir maior fluxo de água
- Pode exigir maior área de contato do aplicador
- Pode demandar ajustes nos ciclos de compressão

### Estratégia

- Não será foco do V1
- Considerado como expansão futura do sistema

## Observações

- O aplicador não é um acessório, é parte do sistema
- Deve ser considerado no projeto térmico e mecânico
- Pode exigir validação específica por região do corpo
