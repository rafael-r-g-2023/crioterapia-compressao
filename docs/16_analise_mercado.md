# ANÁLISE DE MERCADO

## Objetivo

Comparar soluções existentes no mercado e posicionar o projeto.

---

## Segmentos identificados

### Linha médica

Exemplo:
- Game Ready

Características:
- Alta precisão
- Uso hospitalar
- Alto custo

---

### Linha esportiva

Exemplo:
- CryoPush

Características:
- Portátil
- Mais simples
- Menor custo

---

## Comparação

| Característica | Game Ready | CryoPush |
|--------------|-----------|----------|
| Preço | Alto | Médio |
| Complexidade | Alta | Média |
| Portabilidade | Média | Alta |
| Controle | Avançado | Básico |

---

## Estratégia do projeto

O projeto será posicionado entre os dois segmentos:

- Mais acessível que soluções médicas
- Mais robusto que soluções esportivas simples

---

## Objetivo

Desenvolver um sistema:

- Portátil
- Funcional
- Com controle adequado
- Custo reduzido
