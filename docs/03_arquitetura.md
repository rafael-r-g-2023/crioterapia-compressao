# ARQUITETURA DO SISTEMA

## Visão geral

O sistema é composto por módulos independentes que trabalham de forma integrada para fornecer terapia de frio e compressão.

---

## Estrutura geral do produto

PRODUTO V1

- Alimentação
- Controle eletrônico
- Sistema térmico
- Sistema hidráulico
- Sistema pneumático
- Monitoramento e segurança
- Interface do usuário
- Aplicador terapêutico
- Estrutura mecânica

---

## 1. Módulo de alimentação

### Função
Fornecer energia para todo o sistema.

### Componentes
- Fonte de alimentação
- Conversores de tensão
- Proteções elétricas

---

## 2. Controle eletrônico

### Função
Gerenciar todo o funcionamento do sistema.

### Responsabilidades
- Controle de bombas
- Controle de compressor
- Leitura de sensores
- Controle de tempo e modos

---

## 3. Sistema térmico

### Função
Gerar e armazenar o frio.

### Componentes
- Reservatório
- Água e gelo

---

## 4. Sistema hidráulico

### Função
Circular água gelada até o aplicador.

### Componentes
- Bomba de água
- Mangueiras
- Filtro
- Conectores

---

## 5. Sistema pneumático

### Função
Gerar compressão controlada.

### Componentes
- Compressor de ar
- Válvulas
- Linhas pneumáticas
- Sistema de alívio

---

## 6. Monitoramento e segurança

### Função
Garantir funcionamento seguro.

### Sensores
- Temperatura
- Pressão
- Fluxo (opcional)
- Nível (opcional)

---

## 7. Interface do usuário

### Função
Permitir interação com o sistema.

### Componentes
- Botões
- Display
- Indicadores

---

## 8. Aplicador terapêutico

### Função
Aplicar frio e compressão ao corpo.

### Estrutura
- Canal de água
- Câmara de ar
- Estrutura têxtil

---

## 9. Estrutura mecânica

### Função
Suportar e organizar todos os componentes.

### Elementos
- Carcaça
- Suportes internos
- Compartimentos

---

## Observações

- Arquitetura baseada em sistema real existente
- Estrutura modular permite evolução futura
- Projeto focado em protótipo funcional V1
