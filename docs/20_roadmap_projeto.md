# ROADMAP DO PROJETO

## Objetivo

Definir a sequência de desenvolvimento do sistema de crioterapia com compressão, desde a fase atual até o protótipo funcional.

---

## Visão geral

O projeto será desenvolvido em fases, priorizando:

- validação técnica
- simplicidade inicial
- evolução gradual

---

## Fase 1 — Estruturação (Concluída)

### Objetivo
Organizar base do projeto

### Entregas
- [x] Documentação inicial
- [x] Engenharia reversa
- [x] Análise de mercado
- [x] Definição de arquitetura
- [x] Planejamento de testes

---

## Fase 2 — Engenharia reversa experimental (Próxima)

### Objetivo
Levantar dados reais do sistema de referência

### Atividades
- [ ] Montar bancada de testes
- [ ] Medir vazão
- [ ] Medir temperatura
- [ ] Avaliar comportamento dos aplicadores
- [ ] Registrar dados na planilha

### Resultado esperado
- Parâmetros reais do sistema
- Base para dimensionamento

---

## Fase 3 — Validação com especialista

### Objetivo
Validar parâmetros de uso com profissional da área da saúde.

### Profissionais sugeridos

- [ ] Fisioterapeuta
- [ ] Ortopedista
- [ ] Profissional de reabilitação esportiva

### Pontos a validar

- [ ] Temperatura segura de aplicação
- [ ] Faixa de pressão adequada
- [ ] Tempo recomendado de uso
- [ ] Frequência de uso
- [ ] Regiões do corpo (joelho, quadril, etc.)
- [ ] Possíveis riscos e contraindicações

### Atividades

- [ ] Agendar reunião com especialista
- [ ] Preparar perguntas técnicas
- [ ] Registrar respostas
- [ ] Atualizar documentação do projeto

### Resultado esperado

- Parâmetros seguros definidos
- Ajustes necessários no projeto
- Base para evolução futura

### Objetivo
Melhorar desempenho e controle

- O projeto deve evoluir de forma incremental
