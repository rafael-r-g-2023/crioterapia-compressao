# VISÃO GERAL DO PROJETO

## Nome do Projeto
Sistema portátil de crioterapia com compressão pneumática

## Objetivo
Desenvolver um equipamento portátil capaz de aplicar terapia de frio e compressão pneumática para recuperação muscular e pós-operatório, com foco inicial em aplicação para joelho.

## Aplicação
- Recuperação pós-cirúrgica
- Lesões esportivas
- Redução de edema e dor

## Conceito do Produto
O sistema combina dois princípios terapêuticos:
- Crioterapia: circulação de água gelada
- Compressão pneumática: aplicação de pressão controlada

## Estratégia do Projeto (V1)

### Tipo de sistema térmico
- Reservatório com água e gelo (sem refrigeração ativa)

### Portabilidade
- Equipamento transportável (não ultracompacto)

### Aplicador inicial
- Envoltório para joelho

### Nível de complexidade
- Médio (foco em funcionalidade e protótipo)

## Objetivo do V1
- Validar funcionamento do sistema
- Testar integração entre módulos
- Criar base para evolução futura

## Escopo atual
- Arquitetura definida
- Engenharia reversa realizada
- Próximas etapas: fluxo de funcionamento e escolha de componentes
